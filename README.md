# 🍾 Spin Bottle – Truth or Dare Game

🎮 A modern **Spin the Bottle Truth or Dare Party Game** built using **HTML, CSS, and JavaScript**.
Play with friends, spin the bottle, and complete fun, romantic, travel, study, friendship, random, and dark challenges!

---

## ✨ Features

* 🍾 **Spin Bottle Animation**
* 👥 **Multiplayer Mode (2–4 Players)**
* 😀 **Player Avatar / Character Selection**
* 🎯 **Random Player Selection**
* ❤️ **Truth or Dare Choice**
* 🔀 **Questions Shuffle Every Round**
* 🎉 **Confetti Effects**
* 🎵 **Sound Effects**
* 🌙 **Modern Neon Dark UI**
* 📱 **Fully Responsive Design**

---

## 🎮 How to Play

1️⃣ Select the **number of players (2–4)**
2️⃣ Enter **player names**
3️⃣ Choose **avatars**
4️⃣ Click **Spin Bottle** 🍾
5️⃣ Selected player chooses **Truth ❤️ or Dare 🔥**
6️⃣ Complete the challenge and spin again!
